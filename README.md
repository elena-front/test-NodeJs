
## 💰 Планировщик личного бюджета

Приложение для персонального планирования бюджета на месяц с круговыми диаграммами расходов, управлением категориями и системой сбережений.

---

### 1. Авторизация и регистрация
- Страница входа и регистрации (email + пароль) через Supabase Auth
- Каждый пользователь ведёт свой индивидуальный бюджет
- Кнопка Logout на главной странице
- Сброс пароля через email

### 2. Главная страница (Dashboard)
- **Карточка «Общий бюджет»**: общая сумма, остаток, сбережения
- **Сетка круговых диаграмм** по категориям (Recharts):
  - 🔴 Красный сектор — выделенный лимит
  - ⚫ Серый сектор — потрачено
  - Под диаграммой: название, лимит, потрачено, остаток
  - Перерасход визуально подсвечивается
- Обязательные категории: коммунальные, школа/секции, еда, бензин
- Дополнительные: одежда, развлечения, сладости, здоровье
- Клик по диаграмме → модальное окно добавления расхода

### 3. Добавление транзакции
- Модальное окно при клике по категории
- Поле суммы, комментарий (опционально)
- Кнопки «Подтвердить» и «Отмена»
- Предупреждение при превышении лимита
- Автоматическое обновление диаграммы и остатка

### 4. Сбережения
- Карточка с зелёным оформлением
- Автоматический расчёт из неиспользованного остатка
- Возможность вручную перенести средства в сбережения

### 5. Управление категориями
- Добавление новых дополнительных категорий
- Редактирование лимита любой категории
- Удаление категории (только если нет транзакций)
- Обязательные категории нельзя удалить

### 6. База данных (Supabase)
- Таблицы: профили пользователей, бюджеты, категории, транзакции, сбережения
- RLS-политики: каждый видит только свои данные
- Данные сохраняются между сессиями

### 7. Дизайн
- Светлый минималистичный стиль
- Чистый белый фон, мягкие тени, аккуратная типографика
- Адаптивная сетка диаграмм
